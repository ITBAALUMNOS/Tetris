BACK END DOCUMENTATION:

*LED DISPLAY is connected VIA SPI
Connect LOAD to PORTA0
Connect CLK to SCK0
Connect DATA to MOSI0

*BUTTONS
Connect all buttons to PORTB
PORTB0 ->BUTTON UP
PORTB1 ->BUTTON DOWN
PORTB2 ->BUTTON LEFT
PORTB3 ->BUTTON RIGHT
PORTB4 ->BUTTON UP_LEFT
PORTB5 ->BUTTON UP_RIGHT
PORTB6 ->BUTTON DOWN_LEFT
PORTB7 ->BUTTON DOWN_RIGHT

*LEDDisplay
connect DIN to OC6 pin
